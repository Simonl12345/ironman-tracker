IRONHYBRID
Training tracker — powered by Supabase + Claude AI.
Follow the steps below, then paste your keys to connect.
Important: This app must be opened via a web server (GitHub Pages etc), not directly from your Downloads folder.
1
Go to sb.com → New project → give it a name → copy your Project URL and anon public key from Settings → API.
2
In Supabase → SQL Editor → paste and run the setup SQL from the second file provided (setup.sql).
3
In Supabase → Authentication → Providers → make sure Email is enabled. Then create your account below.
4
Paste your keys and sign up. You're in.
Get a free key at console.anthropic.com → API Keys → Create key. Stored only in your browser.
One more Supabase step: In Supabase → Authentication → URL Configuration → add your GitHub Pages URL (e.g. https://username.github.io) to Site URL and Redirect URLs. This allows login to work from your hosted URL.
